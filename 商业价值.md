# 商业价值一页纸 — 不靠拍脑袋的假数，靠美团天天在测的真实指标

> 配合「商业星途大奖」。一句话：**一句话撬动整桌连带消费（客单↑），信任越深越离不开（留存↑）——这两个都是美团每天都在测、都想提的标准指标，我们直接把它们抬上去。**

---

## 为什么换掉"决策放弃率"这类指标

"有多少人因为懒得安排而放弃"——听着性感，但**统计不出、也无法优化**，BD 不会信。
所以我们只讲**真实可测、可优化、且演示里看得见**的两个指标：

| 指标 | 它是什么 | 为什么可信 |
|---|---|---|
| **连带率 / 客单价 (Attach rate / AOV)** | 一次会话触发了几个 SKU、客单多少 | 美团标准指标，天天在测；产品直接抬高 |
| **信任驱动的留存 / 复购 (Retention / LTV)** | 用户越用越授权、越离不开 | 标准电商指标；信任分=可量化的粘性资产 |

---

## 一、连带率：一句话 → 整桌生意（演示里直接看见）

今天用户打开美团，通常**只点一份外卖 = 1 个 SKU**。
我们的产品把一句"帮我安排下午"变成 **餐厅 + 活动门票 + 礼物 + 用车 的连带下单**——

```
演示实测（执行后"商业价值面板"实时算出，不是 PPT 数字）：
  一句话 → 连带 3 个 SKU：活动/门票 + 餐厅 + 礼物
  撬动 ¥564　|　客单 ¥188/人
  对比"只点一份外卖"(约¥50)：单次会话价值 ↑ 11×
```

**关键：这个数是从真实下单算出来的，不是估的。** 连带率/客单价是美团核心北极星之一，
而"一句话撬动多 SKU"正是直接拉它的机制——这部分需求今天大量**因为麻烦而根本没发生**。

> 演示纪律：执行后面板会显示"连带 N 个 SKU、撬动 ¥X、↑ N×"——**当场让评委看见连带在发生**，
> 而不是听你念假设。基线 ¥50 标注"示意"，不冒充财报。

## 二、留存：信任账户把"一次性交易"变成"越用越离不开"

普通工具是**用完即走**；我们做了一条**信任账户**主线：
- 用户每确认一次它的判断 → 信任分 +；
- 信任越高 → 它越敢替你做主（你反复确认过的事直接替你定）；
- **于是它越用越懂你、越用越省心 → 你越离不开。**

**信任分就是一份可量化的粘性资产**——它直接对应留存率和复购周期，这是 LTV 的来源。
别的平台抄得走一个规划功能，**抄不走你和你的 Agent 之间已经攒了 80 分的信任关系**。

---

## 三、护城河：为什么只有美团能把这串闭环跑通

一句话撬动连带，需要**同时**给齐三样，**只有美团生态一家凑得齐**：
1. **全品类本地供给**（餐厅+活动+鲜花蛋糕+用车）——连带的前提；
2. **即时履约网络**——"下午就要"的短时需求，没有履约只是一张空计划；
3. **决策与偏好数据规模**——每次"采纳/拒绝"都是监督信号，喂养信任账户，**越用越准**。

ChatGPT 给得了计划给不了履约；高德给得了地点给不了全品类闭环。**这条护城河是生态级的。**

---

## 四、为什么现在必须做（入口防御战）

"周末怎么过"是家庭最高频、最有粘性的决策入口。**今天没人把它做成"一句话办完"，但它一定会被做成。**
美团若不占住这个"替你做决定"的入口，就会从"用户主动搜索时才被想起的履约方"，
退化成"被别人(抖音本地生活/高德)调用的供给后端"。**这是一场入口防御战，不是锦上添花。**
