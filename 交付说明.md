# 交付说明（提交清单）

> 作品：**DecisionMate** —— 会替你拿主意、且服务整个"局"的本地活动规划执行 Agent
> 全程 **Python 标准库**，零第三方依赖；不联网也能跑（高德/LLM 为可选增强，调不通自动回退本地 Mock + 磁盘缓存）。

## 一、三项交付物对照

| 要求 | 对应内容 | 在哪 |
|---|---|---|
| **1. Demo（命令行 / WebUI 均可）** | 自动演示脚本 + 真人交互 CLI + 图形化 WebUI（含朋友协同分享页） | `run_demo.py`、`agent/cli.py`、`agent/web.py` |

| **3. 设计文档（2 页：Planning 策略 / 工具调用链路 / 异常处理机制）** | 2 页设计文档 | `DESIGN.md` |

## 二、如何运行 Demo

```bash
# ① 命令行自动演示（推荐先看这个，最快理解全貌）
python3 run_demo.py                 # 家庭主场景：带老婆孩子出去玩，老婆减肥
python3 run_demo.py friends         # 朋友局泛化（4 人口味不一，同一套代码）

# ② 真人交互 CLI（用 `as 老婆` 切视角去改方案，模拟协同评审）
python3 -m agent.cli

# ③ 图形化 WebUI（浏览器，零依赖）
python3 -m agent.web                # 自动开 http://127.0.0.1:8000
#   主页流程：输入意图 → 方案确认 → 协同反馈 → 定稿确认（看新方案+出行提醒）→ 执行
#   加要求可循环再生成（自然语言/快捷按钮），满意才执行
#   朋友协同：页内"生成链接"，朋友用自己手机（同 WiFi）打开 /join 提交改动
#   （9 条验收 + 93 项自检走命令行，见下方"五、自检"）
```

### 可选：开启真实数据（高德 POI / LLM）
复制 `agent/llm_config.example.json` 为 `agent/llm_config.json`，填入你自己的高德 key（和可选 LLM key）。**仓库不含任何密钥**；不填则全程走本地 Mock，Demo 照常完整。

## 三、Tool 实现（`agent/tools.py`，全 Mock API）

`ToolBox` 类，返回数据均带齐决策所需字段（人均/儿童友好/低卡/过敏原/距离/排队/坐标）：

| 阶段 | 工具方法 | 说明 |
|---|---|---|
| 查 | `search_restaurants` / `search_activities` | 带约束过滤；真实模式接高德 POI（`amap.py`），失败回退 Mock |
| 订 | `book_restaurant` / `book_activity` / `join_queue` | 可强制"满了/超时" |
| 买 | `order_flowers` / `order_cake` / `buy_ticket` | 可强制"支付失败" |
| 送 | `arrange_delivery` | 可强制"配送失败" |
| 发 | `send_plan` | 产出可分享方案卡（`card.py`） |
| 账务 | `_commit` / `rollback` | 事务记账，保证 GMV 与账本一致 |

可复现性：固定随机种子 `seed=7` + `forced` 强制项，演示稳定、且能主动演异常。

## 四、代码结构总览

```
agent/
  models.py        核心数据结构（Decision/Plan/Edit/Conflict…）含 5 个"灵魂字段"
  rules.py         置信度×代价 → 做/建议/问；餐厅打分；把"谁改的"纳入置信度
  intent.py        意图解析（已知/未知清单、活动/菜系/人数/场景词、否定感知）
  tools.py         ★ 工具层（查订买送发），全 Mock，可造缺失/失败/延迟
  amap.py          高德真实 POI + 定位（全国 POI 搜索、坐标转换、静态地图）
  planner.py       生成第一版 Plan + 内部冲突协调 + 时间线倒排(饭前/饭后) + GMV
  review.py        协同评审：自动合并 + 带权重冲突建议 + 再规划
  execution.py     ★ 执行 + 异常兜底（满了/超时重试/失败回滚）+ GMV 计数
  orchestrator.py  端到端编排（被 demo/cli/web 复用）
  web.py           图形化 WebUI + 朋友协同分享页（标准库 http.server，零依赖）
  card.py          可分享方案卡（自包含 HTML）
  llm.py/llm_tools.py  可选 AI 层：意图解析 + 候选生成（OpenAI 兼容，带缓存+回退）
  selftest.py      内部逻辑自检（93 项断言）
DESIGN.md          ★ 2 页设计文档（Planning/工具链路/异常处理）
check_acceptance.py  命令行验收自检（对照 PRD 9 条"必须演出来的瞬间"）
run_demo.py        Demo 入口
```
